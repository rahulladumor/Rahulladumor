The security of this repository is taken seriously. If you discover a security vulnerability, please follow these steps:

### How to Report

**Please DO NOT open a public issue.** Instead:

1. **Email**: Send details to **contact@acloudwithrahul.in**
2. **Subject**: Use "SECURITY: [Brief Description]"
3. **Include**:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (optional)

### What to Expect

- **Acknowledgment**: Within 48 hours
- **Initial Assessment**: Within 5 business days
- **Resolution Timeline**: Depends on severity
  - Critical: 1-7 days
  - High: 7-14 days
  - Medium: 14-30 days
  - Low: 30-60 days

### Responsible Disclosure

We follow responsible disclosure practices:

1. You report the vulnerability privately
2. We acknowledge and investigate
3. We work on a fix
4. We coordinate disclosure timing with you
5. Public disclosure happens after fix is deployed
