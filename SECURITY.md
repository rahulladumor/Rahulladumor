# Security Policy

## Reporting a Vulnerability

If you discover any security issues in this repository (broken links, exposed information, etc.), please report them responsibly:

**Email:** contact@acloudwithrahul.in

## What to Report

- Broken or malicious links
- Exposed sensitive information
- Security concerns with third-party integrations
- Any other security-related issues

## Response Time

- **Acknowledgment:** Within 48 hours
- **Resolution:** Within 7 days for critical issues

## Scope

This repository contains:
- Public profile information only
- Links to external services (shields.io, GitHub APIs, Credly, etc.)
- No sensitive data or credentials

Thank you for helping keep this profile secure!
